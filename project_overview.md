# Обзор статуса файлов и структуры проекта

## Критерии статуса:
- **Входит в структуру и реализован полностью** — файл содержит рабочую бизнес-логику, модели, сервисы, шаблоны, используемые в проекте.
- **Входит в структуру и реализован частично** — файл содержит только структуру, заглушки, или не все методы реализованы.
- **Не входит в структуру** — файл отсутствует, но заявлен в архитектуре/ТЗ.
